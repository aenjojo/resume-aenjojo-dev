# Introduction
**Wofi**, abbreviation of "**Wo**rd **Fi**nding", is a website I created to find related word to my search.
This website is my first time of creating a Svelte website and my first time learning Svelte.

The website uses [datamuse](https://datamuse.com) API, an API to search related word.
The API itself is hard to use, if used by manually writing the query into the browser.
So, I build this website, just to find a related word from their API.

Previously, I implement this API on my discord bot, [Natsuki](/projects/non-web/natsuki-bot).
But, since my discord bot is offline for rewriting into Discord.js v14, I can't use it.
